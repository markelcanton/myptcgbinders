## myptcgbinders
--------
_myptcgbinders_ es una página web para compartir contenido de álbumes de cartas Pokémon de forma pública. Incluye contenido interactivo y fotos directas de la [base de datos de cartas oficial de Pokémon]([url](https://www.pokemon.com/es/jcc-pokemon/cartas-pokemon)) y enlaces directos a [Cardmarket](https://cardmarket.com/) para buscar su precio.

_myptcgbinders_ no está asociado ni tiene ninguna afiliación con Pokémon, Nintendo, Cardmarket u otras. Este proyecto ha sido desarrollado por Markel Cantón con fines informativos y de interacción. Su creador jamás hará ningún uso comercial del mismo.
